Combinator Selectors

AND Selectors

A CSS selector which selects only elements that contain ALL of the selectors provided

element.class Selector

Selects elements which have the tag of `element`, and the class of `.class`

#id.class Selector

Selects elements which have the id of #id and class of .class

[andSelector="id"].class Selector

Selects elements which have the id of #id and class of .class

[andSelector="id"].class Selector

Selects elements which have the id of #id and class of .class
